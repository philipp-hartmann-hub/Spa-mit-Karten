# Spaß mit Karten

Geografie-Spiel: Länder auf einer spektakulären Weltkarte finden.

## Start

```bash
npm install
npm run dev
```

## Version 1.0

- Interaktive Weltkarte (Natural Earth / TopoJSON)
- Zufälliges Land wird vorgeschlagen (deutsche Namen)
- Klick auf die Karte → Treffer oder Aufdecken
- Punkte, Serie und Rundenanzeige
